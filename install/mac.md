# MacでのROOTのインストール

Mac上でROOTをビルド(コンパイル)します.
難度はやや高めです.

 - XCodeのインストール
AppStoreからXCodeをインストールし, 起動します. 
最初はLicenceの承諾を要求されるので, 認めてください.
![XCodeのインストール](img/mac_1.png)
